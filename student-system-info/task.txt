Задача: Мини-система учёта студентов

Хранение данных:
- Использовать словарь для хранения информации о студентах.
- Для каждого студента хранить:
  имя (строка),
  возраст (целое число),
  оценки (список целых чисел),
  факультет (кортеж из двух строк: университет и факультет),
  уникальный идентификатор (множество, содержащее одно целое число).

Добавление студентов:
- Написать функцию add_student, которая добавляет нового студента в словарь.
- Аргументы: имя, возраст, список оценок, факультет (по умолчанию ("Unknown", "Unknown")), ID (по умолчанию генерируется автоматически).
- Функция должна возвращать обновлённый словарь студентов.

Средний балл:
- Написать функцию average_grade, которая вычисляет среднюю оценку конкретного студента.
- Использовать лямбда-функцию совместно с map или filter.

Вывод информации:
- Написать функцию print_student_info, которая выводит информацию о студенте.
- Функция должна поддерживать вызов с именованными аргументами, например:
  print_student_info(name="Alex", students=students)

Изменяемость и неизменяемость:
- Продемонстрировать работу изменяемых и неизменяемых типов:
  изменить список оценок студента (операция должна пройти успешно),
  попробовать изменить кортеж факультета (должна возникнуть ошибка).

Динамическая типизация:
- Создать переменную data = 10,
- затем присвоить ей строку data = "Hello",
- вывести оба результата.
